Video360 · behavior + audio + sentiment

A pipeline that turns a 360° meeting recording + 4 lapel-mic recordings into one synchronized web view: per-participant cropped video, YOLO-tracked behavior (head pose, gaze, note-taking), Whisper transcription with bleed-suppressed speaker attribution, and per-utterance sentiment plotted on Russell's valence/arousal circumplex.

YOLOv8 detect+track MediaPipe face/pose FIR decrosstalk Dominant-channel diarization mlx-whisper turbo DistilRoBERTa emotion → V/A
Open the unified app Source on GitHub

Note: this hosted page is code only. The live demo needs the user's own meeting recording (mp4 + 4-channel wav) — see "Run locally" below.

video pipeline
  1. 1
    Auto-crop participants
    YOLOv8 person detection on 4 panorama slots → fixed crop windows → 4× person_slot{0..3}_640x900.mp4
  2. 2
    Pose + face features
    YOLO pose + MediaPipe face mesh → yaw/pitch proxies, smile score, wrist motion per frame
  3. 3
    Behavior inference
    Threshold + temporal logic → look_down, turn_left/right, head_nodding, smiling, take_notes flags
audio pipeline
  1. 1
    Decrosstalk
    Per-channel FIR fit on "target silent" frames; subtract predicted bleed. ~−18 dB cleaner.
  2. 2
    Dominant-channel merge
    Loudest channel wins each frame (hysteresis + Hann crossfade) → single mono + speaker labels
  3. 3
    Whisper ASR per channel
    mlx-whisper-large-v3-turbo on each clean wav; merged dialogue with speaker attribution
  4. 4
    Sentiment → V/A
    DistilRoBERTa-emotion (7 classes) → weighted Russell anchors → 4 quadrants + 13 fine labels
run locally

Bring your own recording

The hosted page can't include real meeting recordings (privacy). To see the live UI with synced panorama / cropped videos / per-speaker audio / transcript / circumplex, run the pipelines on your own data: 

# 1. clone + venv
git clone https://github.com/Anny405/video360_video_audio_sentimentAnalysis.git
cd video360_video_audio_sentimentAnalysis
python3 -m venv .venv && source .venv/bin/activate
pip install -r dist/requirements.txt
pip install mlx-whisper transformers soundfile

# 2. drop your own recordings
#    - test_run/test_input.mp4                       (panorama)
#    - test_run/audio/*.wav                          (per-mic, one per speaker)

# 3. video chain
python dist/00_autocrop_people.py --input test_run/test_input.mp4 --out test_run/autocrop_out
python dist/01_yolo_detect_track_hpc.py --input-dir test_run/autocrop_out --output-dir test_run/features_yolo01
python dist/02_face_headpose_speaker_notetaking.py --input-dir test_run/features_yolo01 --output-dir test_run/behavior_outputs --no-html

# 4. audio chain (run from inside test_run/audio)
python decrosstalk.py *.wav
python merge_dominant.py *_clean.wav -o merged_demo --names S1 S2 S3
python transcribe.py *_clean.wav --merge -o transcripts
python sentiment.py

# 5. build demo data + serve
python mockups/build_data.py
python mockups/build_audio_data.py
python mockups/serve.py 8765
# open http://127.0.0.1:8765/mockups/app.html
